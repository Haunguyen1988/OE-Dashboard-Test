# Plan: Live Flight Tracking — OpenSky Network API

Created: 2026-02-26
Status: 🟡 In Progress

## Overview
Tích hợp OpenSky Network REST API vào Network Map View của OE Dashboard. Thay thế placeholder dots bằng bản đồ thật (Leaflet) hiển thị vị trí real-time máy bay VJ, kết hợp dữ liệu flight từ Supabase.

## Tech Stack
- **Frontend:** React 19 + TypeScript + TailwindCSS 4 (existing)
- **Map:** Leaflet + React-Leaflet (NEW)
- **Tiles:** CartoDB Dark Matter (free, dark theme)
- **API:** OpenSky Network REST API (free tier)
- **Data:** Supabase (existing)
